## Installation

1. Clone the repository:

```
 git clone git@github.com:NatJaden/react-phase2-group-project.git
```

2. Install dependencies:

```
 npm install
```

3.  Run the back-end:

```
 json-server --watch db. json
```

4.Run the front-end:

```
npm run dev
```
